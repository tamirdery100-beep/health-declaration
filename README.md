# הצהרת בריאות - Alya Berkovich

טופס דיגיטלי להצהרת בריאות לטיפול קוסמטי.
HTML/CSS/JS סטטי - ללא backend - שליחה דרך EmailJS.

## תכונות
- שאלון בריאות מקיף בעברית (RTL)
- שדות פירוט דינמיים לתשובות "כן"
- חתימה דיגיטלית במצב Focus (מודאל מלא)
- לוגיקת גיל: מתחת ל-18 דורש חתימת הורה
- ולידציה מלאה לפני שליחה
- אופטימיזציה למובייל (iOS Safari, Android Chrome)
- אינטגרציה עם EmailJS

## קבצים
- `health-declaration.html` - הקובץ הראשי (כל הלוגיקה והעיצוב)
- `index.html` - העתק לצורך נתיב ברירת מחדל
- `logo.png` - לוגו הקליניקה

## פריסה
מתאים ל-Vercel / Netlify / GitHub Pages - אין שלב build, רק קבצים סטטיים.
